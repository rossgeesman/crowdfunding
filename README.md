Crowdfunding App
================

Details
-------

Crowdfunding site built for Freshist. Also my first Rails app. Sample up at https://crwdfndr.herokuapp.com

Environment
-----------

* Ruby 2.0.0 & Rails 4.0.0
* PostgreSQL 9.3.4
* DelayedJob for background processing
* Stripe Payments API
* Amazon S3
* Heroku
* Bootstrap
